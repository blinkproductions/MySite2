Xtraction

Using Epic Games free Paragon assets I was able to create a thrilling third person shooter game. The goal of the game is to eliminate all the enemies and to find and extract the objective in a location.

This was implemented using C++ and blueprints. I then created a simple AI that guards a position using AI controllers, Behavior Tree, and a Blackboard. The player's character movement animations were created by using animation blendspaces. I used line traces for the gun which has a modifiable range to adapt for future changes if I decide to make a new weapon such as a pistol.

Gone Clubbin'

A demonic army from hell has started its invasion of Earth in the sprawling golf courses of central Florida. The only thing standing between humanity and complete annihilation is an incredibly senile Florida granny who will stop at nothing to chase every single demon off her lawn.

This was a game jam submission I joined were my team was chosen to represent our university at GameSIG. There are different types of golf balls here that I implemented that had different types of actions and reactions. For example, a grenade golf ball, homing missile golf ball, a ball that can make you teleport, and of course a normal golf ball. I also implemented the user interface using Unreal's UMG UI Designer.
